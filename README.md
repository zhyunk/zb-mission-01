(1) 테이블이 포함된 ERD파일(이미지 캡쳐 화면도 추가로 같이 제출)

- 프로젝트 소스는 Github 로 제출 (제출 방법에 대한 안내는 특강형식으로 따로 제공될 예정)

1) 본 페이지 하단에 위치한 과제 제출 버튼 클릭

#### ✅ (1) 제로베이스 백엔드 스쿨_N기_이름(실명)_\[마스터반\] Mission 1_날짜(230101) 제출 완료라는 메세지 작성
